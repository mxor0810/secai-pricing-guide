## 色采能干什么？先说说它的真实使用场景

色采的核心功能是「相机实时配色」，打开之后摄像头画面里的颜色会被自动抓取，移动手机就能不断刷新色卡组合，看到喜欢的点一下锁定，完事儿。

除此之外还有几个场景蛮实用：

**照片取色**：不是实时拍的，相册里那张好看的旧照片也能用。打开图片，算法自动识别主色调，生成色卡，不用手动吸色。

**灵感色卡库**：高级版里有 800+ 张原创色卡，从故宫宝藏的传统色到西方油画里的配色，从自然风光到影视动漫——说白了就是帮你打开眼界，让你知道"原来这两个颜色可以放在一起"。
